## 🧩 Engineering Focus

- 단순 구현보다 **수집 → 저장 → 분석 → 출력 → 검증**까지 이어지는 흐름을 중요하게 봅니다.
- 추천이나 점수화 로직은 결과만 보여주지 않고, 왜 그렇게 판단했는지 설명 가능하게 만듭니다.
- 프로젝트를 코드에서 끝내지 않고, 포트폴리오 케이스스터디로 정리해 구조와 의사결정을 남깁니다.
