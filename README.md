## 🚀 Featured Projects

### 🔐 GateGuard — AI Web Access Control Security System  
👉 https://github.com/tami-bang/GateGuard

**Why**
- 기존 웹 보안 시스템은 정적 정책 기반 → 새로운 위협 대응 한계
- AI + 정책 기반 구조로 개선 필요

**What I Built**
- C 기반 HTTP 패킷 탐지 엔진 (libpcap, raw socket)
- 정책 기반 → AI fallback 구조 설계
- HTTP 403 Injection + TCP RST 차단
- FastAPI AI Scoring 서버 (평균 응답 ~2.7ms)
- MariaDB 로그 구조 설계 (access_log, ai_analysis)
- Admin UI (Next.js) + 실시간 보안 이벤트

**Impact**
- 약 **19,000+ HTTP 요청 처리 및 분석**
- 차단 응답 **0~1ms 수준으로 전송 (race condition 대응)**
- 정책 기반 우선 처리로 **AI 호출 최소화 및 성능 개선**

**Key Skills**
- 네트워크 패킷 처리 / Raw Socket
- 정책 → AI fallback 설계
- race condition (403 vs 200) 해결
- 로그 기반 디버깅 및 구조 개선

### 🏥 health-ai-search-api — Healthcare AI Search Service  
👉 https://github.com/tami-bang/health-ai-search-api

**Why**
- 단순 키워드 검색은 의료 정보 정확도 부족
- 의미 기반 검색 + 요약 필요

**What I Built**
- 증상 입력 → retrieval → semantic rerank → LLM 요약
- 다국어 처리 및 응답 localization
- fallback 구조 설계 (API 실패 대응)

**Impact**
- 검색 결과 품질 개선 (rerank 적용)
- 응답 생성 파이프라인 구조화
- 사용자 입력 기반 의미 검색 구현

**Key Skills**
- NLP 기반 검색 구조 설계
- Retrieval + Rerank + LLM Pipeline
- FastAPI 서비스 설계
